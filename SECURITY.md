# Security Policy

## Supported Versions

Only the latest public release is supported.

## Reporting a Vulnerability

Please open a private security advisory on GitHub if you find a vulnerability.

If that is not available, open a GitHub issue with a high-level description and avoid posting exploit details publicly until there is a fix.

## Distribution Notes

Community DMG builds may be ad-hoc signed and not Apple-notarized. For the strongest install experience, build locally from source or use a Developer ID signed and notarized release.
